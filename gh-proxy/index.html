GitHub 文件加速下载

快速下载GitHub上的文件、Release和Archive资源

使用说明

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都符合标准。

注意：不支持项目文件夹的下载。

合法输入示例：
  • https://github.com/owner/repo/blob/main/file.zip
  • github.com/owner/repo/releases/tag/v1.0.0
  • raw.githubusercontent.com/owner/repo/main/image.png
